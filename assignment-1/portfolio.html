Portfolio

Here's a list of all my notable projects so far:


Restaurant Reservation System Frontend

See the project on GitHub

This is my first non-academic related project. I created it as a proposal for a potential replacement of a restaurant's pen/paper reservation system. Built with React Native and Expo CLI. This is the frontend to a fully functional reservation system. More information can be found on the GitHub project page.


Restaurant Reservation System Backend

See the project on GitHub

Built using MongoDB, Express.js and Node.js. This project served as further learning and experience with the MERN stack. This project is also fully functional and ready to use.


Microsoft Edge - Chromium Addon : Modal Reader

See the project on the Edge Addon Store

This is a small chromium extension I created to help me read poorly formatted documentation or any large chunks of text on the web.


COMP 3100 Final Project : Human Height Visualizer

See the project on GitHub

A final project submission for my COMP 3100 Web Development course at Memorial University of Newfoundland. This was my first introduction to web development on a whole as well as the Node.js, Express.js and MongoDB stack. Basically it takes a dataset of heights across all countries and visualizes it using graphics.


COMP 4302 Final Project : 3d WebGL Flappy Bird Clone

See the project on GitHub

Another final project submission for my COMP 4302 3D graphics course at Memorial University of Newfoundland. This was my first introduction to the WebGL landscape as well as programming with the modern openGL pipeline. It also uses the Entity Component Systems (ECS) concepts learned in my Game Development course.
